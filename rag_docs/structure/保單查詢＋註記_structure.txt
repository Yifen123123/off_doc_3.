【文件用途】
本說明文件用來描述「保單查詢＋註記」類型公文的基本結構與應擷取欄位。
此類公文係同時包含：
1. 保單查詢（如：契約資料、變更紀錄、給付紀錄、借款紀錄、解約金等）
2. 保單註記（如：禁止變更要保人、禁止解約、禁止借款、禁止給付等）

本類型公文主要出現在：
1. 法院對保險契約進行查詢並要求保全；
2. 消費者債務清理案件（更生／清算）中，法院需查詢保單資訊並禁止債務人處分；
3. 金融機構、壽險公會等機關函轉法院來文。

公文可能同時包含查詢與禁止處分兩類資訊，需完整擷取。

--------------------------------------------------------------------------------
【典型整體結構】
保單查詢＋註記公文通常包含下列區塊（格式與順序可略有差異）：

1. 抬頭與發文機關
2. 聯絡資訊（地址、承辦人、電話、傳真）
3. 受文者
4. 發文日期與發文字號
5. 主旨（說明查詢事項與註記事項）
6. 說明（一般以一、二、三條列）
7. 若為函轉，會包含函轉來源／原發文機關等資訊

--------------------------------------------------------------------------------
【共通欄位說明】
以下欄位於所有「保單查詢＋註記」公文中皆應擷取，且與其他公文類型共通：

1. 發文機關  
   例如：「臺灣○○地方法院民事執行處」、「中華民國人壽保險商業同業公會」等。

2. 發文日期  
   明確以「發文日期：中華民國○年○月○日」標示。

3. 發文字號  
   以「發文字號：○○○○字第○○號」表示。

4. 承辦資訊  
   包含承辦人姓名、聯絡電話、傳真等。

5. 受文者  
   通常為「○○人壽保險股份有限公司」。

6. 查詢對象（姓名與身分證字號）  
   常以「債務人○○○（身分證統一編號：X123456789號）」方式出現。

--------------------------------------------------------------------------------
【一、保單查詢類專屬欄位說明】

公文通常會要求查詢數項保險相關資料，主要可分類如下：

-------------------------------------------------------------------------------
（一）契約變更項目  
常見用語：
- 「保單變更資料影本」
- 「所有保單變更資料」
- 「契約變更項目及日期」

例如：  
「請提供該保單於起訖期間內之所有契約變更資料。」

需擷取：
- 是否提及契約變更  
- 項目（如：受益人變更、繳別變更、保額變更等）  
- 變更日期  
- 變更前後資料（如公文有揭示）

-------------------------------------------------------------------------------
（二）要保人變更  
常見用語：
- 「保單若曾變更要保人，請說明變更日期」
- 「要保人是否曾變更及相關紀錄」

需擷取：
- 是否曾變更  
- 變更日期  
- 變更前後要保人（如有）

-------------------------------------------------------------------------------
（三）保險金給付紀錄  
常見用語：
- 「是否近2年內請領失能或傷病保險金」
- 「請提供所有保險金給付紀錄」
- 「如有領取解約金、生存金、滿期金等，請敘明時間與金額」

需擷取：
- 是否查詢給付紀錄  
- 給付種類（失能、傷病、生存金、滿期等）  
- 給付期間（如原文有載）  

-------------------------------------------------------------------------------
（四）保單借還款紀錄  
常見用語：
- 「如有保單借款，請並敘明借款金額、日期及餘額」
- 「請提供保單借款及還款明細」

需擷取：
- 是否查詢借款紀錄  
- 借款金額  
- 借款日期  
- 是否已清償  
- 扣除借款後之解約金  
- 還款情形

-------------------------------------------------------------------------------
（五）解約金資訊  
常見用語：
- 「如於文到當日解約，請提供解約金試算」  
- 「扣除保單借款後尚餘之金額」

需擷取：
- 解約金金額  
- 扣除借款後金額  
- 是否要求提供「變更前後解約金」

--------------------------------------------------------------------------------
【二、保單註記類專屬欄位說明】

此類公文核心功能為禁止債務人處分保單（保全處分）。
常見註記項目如下：

1. 禁止變更要保人  
2. 禁止解約或終止契約  
3. 禁止保單借款  
4. 禁止設定負擔或移轉（如質押）  
5. 禁止為其他減損保單價值之行為  
6. 禁止第三人（保險公司）支付保險給付  

需擷取：
- 是否逐項禁止  
- 是否有例外情形（例如：只禁止主約，不禁止附約等）

--------------------------------------------------------------------------------
【三、函轉型公文之特殊欄位說明】

若公文為函轉形式（例如壽險公會函轉法院函），需額外擷取：

（一）函轉來源  
- 原發文機關  
- 地址、承辦人、聯絡方式  

（二）最終應回覆機關  
公會常寫明：
- 「請逕復法院」  
- 「毋須副知本會」

（三）本案最終要求查詢之內容  
多存在於原法院函（附件中之說明二、說明三）

-------------------------------------------------------------------------------
【判讀原則】
1. 四項查詢類別（變更／要保人／給付／借款）只要原文有提及，即視為需查詢。  
2. 註記類別同理，以原文明載為準，不可臆測。  
3. 函轉案需優先以「原法院來函」內容為主。  
4. 未載明之資訊請填「無明確記載」。

--------------------------------------------------------------------------------
